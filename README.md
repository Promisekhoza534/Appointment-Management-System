# Appointment-Management-System
A simple vb.net application for managing appointments in local clinic developed as part of my ICT Application Development diploma.
It allows users to REGISTER, LOGIN, dReset password using otp

## Technologies Used
- vb.net
- Access

## Features
- Add User details
- Login using registered credentials 
- book an appointment


## How to Run
1. Open the project in vb.net
2. Run the main class
